## Installation

```console
$ pip install -U git+https://github.com/Dink4n/wi-mote.git
```

## Usage

```console
$ wi-mote
```

This will launch a qr-code in the terminal, scan the qr-code in
the device you want to use as the remote. Enjoy :)

`ctrl + c` to exit the server.
